# ReadWrite

Scan handwritten or printed notes with AI. Upload photos, get a summary and full transcription instantly.

**Live app**: https://sapeur49.github.io/note-scanner/

## How to Use

1. Open the app and enter your [Anthropic API key](https://console.anthropic.com/) — it's saved in your browser for next time
2. Upload one or more photos of your notes (drag-and-drop, file picker, or camera)
3. Tap **Scan Notes**
4. Get a summary and full transcription — share via your device's share sheet or copy to clipboard

## Features

- Multiple images processed in one pass
- Summary first, full transcription below
- Share summary, transcription, or both via native share sheet (or clipboard on desktop)
- Works on mobile and desktop
- No account required — uses your own Anthropic API key

## Privacy

Images are sent directly from your browser to Anthropic's API. Nothing is stored on any server. Your API key is saved only in your browser's localStorage.
